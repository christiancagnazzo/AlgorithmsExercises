"make all" to compile all
"make clean" to delete all executable file
"bin/test_insertion_sort" to run test for insertion sort algorithm
"bin/test_quick_sort" to run test for quick sort algorithm
"bin/app < -1 | -2 | -3 > < -A | -B | -C > <filename>" to run app

  -1: sort through the insertion sort algorithm
  -2: sort through the quick sort algorithm
  -3: sort through the quick sort three way algorithm

  -A: sort according to the first field ascending order
  -B: sort according to the second field ascending order
  -C: sort according to the third field ascending order
